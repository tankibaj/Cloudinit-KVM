To view all these commands on Terminal, run:: egrep '^\s+\$' GettingStarted.txt | sed -e 's@\$@@'


Commands::

    # Cloudinit configuration
    $ nano user-data.yml
    # Network configuration
    $ nano network-config.yml
    # Virtual machine configuration
    $ nano build.sh
    # Snip Up VM
    $ ./build.sh
